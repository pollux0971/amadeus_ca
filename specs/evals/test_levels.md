# Test Levels

## Level 0. Structure Test

檢查 repo 與 skill package 結構是否完整。

## Level 1. Schema Test

檢查 YAML / JSON 是否符合 schema。

## Level 2. Unit Test

測試單一 function 或 script。

## Level 3. Skill Test

測試單一 skill package 能否獨立執行。

## Level 4. Integration Test

測試多個 skills 串接。

## Level 5. End-to-End Test

測試完整 agent harness 流程。

## Level 6. Adversarial Test

測試 prompt injection、secret leak、dangerous command。

## Level 7. Regression Test

比較 baseline 與 candidate 是否退步。
