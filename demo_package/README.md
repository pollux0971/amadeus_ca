# Project Demo Package (v0)

Single entry point for understanding and demonstrating this project: what it is,
what's done, how to run a safe demo, how to view the dashboard, the safety
boundaries, and the next bounded stories.

> **Everything here is read-only / safe to show.** No real API call, no secret
> read, no stable promotion, no raw shell. All forward work is a **bounded story**
> (see `docs/epics/`).

4. [`04_dashboard_demo.md`](04_dashboard_demo.md) — the read-only dashboard.
