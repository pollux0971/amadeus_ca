# Story UI-V0 — UI dashboard planning only

**Epic:** EPIC-UI
**Status:** done (planning only) — see
[`../../../reports/story_ui_dashboard_v0/README.md`](../../../reports/story_ui_dashboard_v0/README.md)
and [`../../ui_dashboard/`](../../ui_dashboard/).

## Goal

Plan a future read-only dashboard over the harness (phases, gates, evals,
candidates, runs, reports) — **planning only**, no UI implementation.

## Scope

- Write the dashboard **information architecture** (what views/panels exist, what
  each shows, from which existing artifacts).
- Write the **redacted-artifact model** (how `runs/`/reports are exposed, all via
  `src/llm/redaction.py`; raw artifacts never shown).
- Write a **future eval list** for when the UI is eventually built.

## Out of Scope

- Any actual web/app code (beyond existing `apps/` placeholders).
- Any action execution, promotion, or live control surface.

## Preconditions

- Read-only access to existing `runs/`, `reports/`, `docs/checkpoints/`, `evals/`,
  candidate status docs. (No new data is generated by this story.)

## Implementation Boundaries

- May write only planning docs (under `docs/` and/or `apps/.../*.md`).
- May **not** add executable UI code, write actions, or any privileged path.

## Acceptance Criteria

- [ ] dashboard information architecture written
- [ ] redacted artifact model written (read-only, redacted; no raw artifacts)
- [ ] no action execution (design is read-only)
- [ ] no raw shell (no UI-triggered commands in the design)
- [ ] no secret display (only redacted values)
- [ ] no promotion from UI (any action routes through existing approval-gated scripts)
- [ ] future eval list created

## Forbidden Zone

- **UI must not trigger raw shell** or any direct command.
- **UI must not display secret**; only redacted artifacts.
- **UI must not directly promote / apply / merge / stage**; actions route through
  existing approval-gated scripts.
- No real API. No stable modification. No secret in any artifact.

## Required Validation Commands

```bash
python scripts/validate_structure.py
python scripts/validate_workflows.py
python scripts/check_secret_hygiene.py
python scripts/run_unit_tests.py
```

## Artifacts to Produce

- Planning docs: information architecture, redacted-artifact model, future eval
  list (all redacted; no secret).

## Rollback / Stop Condition

- **Rollback:** delete the planning docs — nothing runtime changed.
- **Stop Condition:** stop when the Definition of Done is met; do not begin any UI
  implementation or continue into another story.

## Definition of Done

Acceptance criteria met; validation green; planning docs exist; working tree clean;
**stop** (no UI built).
